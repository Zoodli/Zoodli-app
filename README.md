# Zoodli | زودلي

Zodli is a collaborative mobile app that helps groups organize restaurant orders together.

Instead of chaotic conversations and mistaken orders, Zodli allows people to:

- add items to a shared session
- split shared dishes easily
- calculate taxes and delivery automatically
- see exactly what each person owes

Built with:
- Kotlin Multiplatform
- Compose Multiplatform
- Supabase
